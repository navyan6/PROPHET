================================================================================
OUTPUT FORMAT
================================================================================

JSON with per-variant and weighted binding scores:

{
  "sequence": "KVMDFSDPFCVEY",
  "binding_per_variant": {
    "var_0": 6.32,
    "var_1": 5.89,
    "var_2": 6.12
  },
  "weighted_binding": 6.11,
  "mean_binding": 6.11
}

weighted_binding = sum(variant_affinity * probability) across all variants
